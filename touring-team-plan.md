## 1. 비전

**"몽골의 안정적 기반 위에서, 한국 ITF 대회 경험으로 세계적 선수를 키운다"**

## 3. 왜 지금인가?

### 아들 (2010년생, 16세)
- 팔꿈치 부상 → 수술 예정
- 수술 후 재활 3~4개월 → **2026년 하반기부터 대회 복귀 목표**
- 16세 = ITF 주니어(U18) 본격 진입 시기
- **지금 시합 경험을 쌓지 않으면 타이밍을 놓침**

### 딸 (2012년생, 14세)
- 선수로 성장하려면 국제 시합 경험 필수
- 몽골 내 대회 부족 → 한국 ATF/ITF가 최적
- 14세 = U14 → U16 전환기, **골든 타임**

### 몽골 사업
- 2년간 시스템 구축 완료
- 앱 기반 관리 = **원격 운영 가능**
- 직원 체계 안정 (체육관 4명, 피트니스 12명, 샵 2명)

## 4. 투어링팀 구조

### 팀 구성
| 역할 | 인원 | 비고 |
|------|------|------|
| 감독/코치 | 1명 | 재성님 (직접) |
| 선수 | 3~5명 | 자녀 2명 + 몽골 유망주 1~3명 |
| 총원 | 4~6명 | |

### 운영 모델
```
[몽골 본거지] ←→ [한국 원정]

몽골 (평상시):
  - 3면 코트에서 정규 훈련
  - 세계적 프로그램 (월~금 3시간)
  - 체력/멘탈/기술 강화

한국 (원정 시):
  - ITF/ATF 주니어 대회 참가
  - 2주 단위 원정 (대회 기간)
  - 대회 사이: 한국 코트에서 훈련 (J-One 활용 가능)
```

### ❌ 예전과 다른 점

| 과거 (제주도) | 현재 (투어링팀) |
|--------------|----------------|
| 한국에 올인, 몽골 포기 | 몽골 유지 + 한국 원정 |
| 코트 임대/건설 비용 | 원정 숙박비만 (or J-One) |
| 협회와 상시 갈등 | 대회 참가만, 정치 무관 |
| 수입원 없이 지출만 | 몽골 3개 사업 수입 유지 |
| 무기한 체류 | 2~3주 단위 원정 후 복귀 |
