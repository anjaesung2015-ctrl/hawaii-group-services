# 🤝 재활센터 협력 제안서
## Hawaii Sports Center × 재활센터 파트너십

---

## 1. 제안 기관 소개

**Hawaii Sports Center (하와이 스포츠 센터)**
- 울란바토르 소재 종합 스포츠 시설
- 🎾 테니스 코트 4면 (실내 3면 + 2층 1면)
- 💪 피트니스 센터 (400+ 회원)
- 🏆 테니스 아카데미 (주니어 선수반 운영)
- 직원 19명 / 대표: 안재성

**왜 재활센터와 협력하는가?**
- 스포츠 선수의 부상 예방 및 재활 수요 증가
- 피트니스 회원의 전문 재활 니즈
- 종합 스포츠 헬스케어 서비스 구축 목표

---

## 2. 협력 모델 제안 (3가지)

### 모델 A: 코치 파견형
> 재활센터에서 전문가를 하와이 센터로 파견

| 항목 | 내용 |
|------|------|
| **방식** | 재활 전문가가 하와이 센터 2층에서 근무 |
| **시간** | 주 3~5일, 하루 2~4시간 |
| **대상** | 피트니스 회원 + 테니스 선수 + 외부 고객 |
| **수익 배분** | 재활센터 60% : 하와이 40% (장소 제공) |
| **하와이 제공** | 공간, 시설, 고객 유입, 마케팅 |
| **재활센터 제공** | 전문 인력, 기술, 장비 (일부) |

**장점:** 하와이 센터 내 원스톱 서비스 → 회원 만족도 ↑

---

### 모델 B: 선수 전담 트레이닝형
> 테니스 선수반 훈련 전후 재활/컨디셔닝

| 항목 | 내용 |
|------|------|
| **방식** | 재활 전문가가 선수 훈련 전후 방문 |
| **스케줄** | 월/수/금 14:00~15:00 (훈련 전) + 19:00~20:00 (훈련 후) |
| **대상** | 테니스 선수반 (주니어 선수들) |
| **비용** | 선수 1인당 월정액 or 회당 정액 |
| **프로그램** | 워밍업, 쿨다운, 부상 예방, 재활 운동, 테이핑 |

**장점:** 선수 부상 예방, 퍼포먼스 향상 → 아카데미 가치 ↑

---

### 모델 C: 공간 공유 + 수익 분배형
> 하와이 센터 2층을 재활 전용 공간으로 제공

| 항목 | 내용 |
|------|------|
| **방식** | 2층 공간 일부를 재활센터가 운영 |
| **하와이 제공** | 공간 (월세 무료 or 저렴), 고객 소개 |
| **재활센터 제공** | 장비 설치, 전문 인력, 운영 |
| **수익 배분** | 매출의 30:70 (하와이:재활) or 40:60 협의 |
| **계약 기간** | 6개월 시범 → 연장 |
| **해지 조건** | 2개월 전 통보 |

**장점:** 초기 투자 최소, 리스크 분산, 양측 윈-윈

---

## 3. 기대 효과

### 하와이 센터
- 🏥 종합 스포츠 헬스케어 시설로 업그레이드
- 💰 신규 수익원 (재활 서비스 수수료)
- 👥 피트니스 회원 유지율 향상 (차별화)
- 🎾 선수 관리 전문성 강화 → 아카데미 브랜드 가치 ↑
- 📈 마케팅 시너지 (재활센터 고객 → 피트니스/테니스 유입)

### 재활센터
- 📍 스포츠 시설 내 거점 확보 (고객 접근성 ↑)
- 👥 400+ 피트니스 회원 + 테니스 수강생 = 즉시 고객
- 💰 추가 매출 (기존 센터 + 하와이 지점)
- 🏆 스포츠 재활 전문 브랜딩

## 4. 운영 세부 사항

### 공간
- **위치:** 하와이 스포츠 센터 2층
- **면적:** 약 ___㎡ (협의)
- **설비:** 기본 전기/수도/냉난방 제공
- **인테리어:** 재활센터 측 부담 (or 공동)

### 타겟 고객
| 구분 | 예상 인원 | 이용 빈도 |
|------|----------|----------|
| 피트니스 회원 (재활 필요) | 월 20~30명 | 주 1~2회 |
| 테니스 선수반 | 10~15명 | 주 3~5회 |
| 일반 테니스 수강생 | 월 10~15명 | 주 1회 |
| 외부 고객 | 월 10~20명 | 수시 |

### 예상 매출 (월)
| 항목 | 단가 | 건수 | 월 매출 |
|------|------|------|--------|
| 재활 1회 세션 | ₮30,000~50,000 | 80~120건 | ₮3~5M |
| 선수 월정액 | ₮100,000~150,000 | 10명 | ₮1~1.5M |
| 패키지 (피트니스+재활) | ₮250,000 | 15명 | ₮3.75M |
| **월 합계** | | | **₮7.75~10.25M** |

## 5. 시범 운영 계획

### Phase 1: 시범 (1~3개월)
- 주 2~3일 전문가 파견
- 테니스 선수반 우선 적용
- 피트니스 회원 대상 체험 프로그램
- 고객 반응 및 수요 파악

### Phase 2: 확대 (4~6개월)
- 주 5일 상주 운영
- 프로그램 다양화 (재활, 교정, 퍼포먼스)
- 패키지 상품 출시
- 공동 마케팅

### Phase 3: 정착 (7개월~)
- 장기 계약 체결
- 장비 투자 확대
- 전담 인력 배치
