## 재성님 일일 서포트

### 체크 항목 (하트비트마다 시간 확인 후 실행)

1. **아침 브리핑** — ❌ 비활성화 (재성님 요청 2026-04-22)

2. **이동 알림** — ❌ 비활성화 (재성님 요청 2026-04-21)

3. **화요일 직원 미팅 준비 (13:00~13:30)** — 화요일이면:
   - 이번 주 매출 현황 (각 사업장별)
   - 직원 업무 완료율 체크
   - 미팅 의제/토론 포인트 생성
   - 다음 주 중요 일정 공유

4. **금요일 주간 리뷰 (10:00~10:30)** — 금요일이면:
   - 이번 주 매출 요약 (전 사업장)
   - 다음 주 스케줄 미리보기
   - 특별 이벤트/마감 알림

### 시간대별 동작 (Asia/Ulaanbaatar = UTC+8)
- 01:00~09:00 UTC (09~17 UB): 활성 시간 — 브리핑/알림 가능
- 09:00~12:00 UTC (17~20 UB): 레슨 시간 — 이동 알림만
- 12:00~16:00 UTC (20~24 UB): 저녁/혼자시간 — 긴급만
- 16:00~01:00 UTC (24~09 UB): 수면 — HEARTBEAT_OK

### 타임블록 데이터
- 파일: `weekly-timeblocks.json`
- 요일별 30분 단위 블록
- 카테고리: 레슨/사업/건강/성장/가족/관계/식사/이동/휴식/나만의시간

5. **방카르 예방접종 알림 (4/14)** — 4월 14일에:
   - "내일(4/15)부터 방카르 예방접종 가능! 동물병원 예약하세요 🐕💉"

6. **엑셀 자동 입력 (매 하트비트)** — 새 엑셀 파일 감지 시:
   - `memory/excel-processed.json` 에서 이미 처리한 파일 목록 확인
   - `~/.openclaw/media/inbound/` 에서 새 xlsx 파일 스캔
   - **피트니스** 파일 패턴: `4.숫자` 또는 `4-숫자` (예: 4.13, 4-13) — Sheet1 기준
   - **센터** 파일 패턴: `04.숫자` (예: 04.13) — Sheet2 기준
   - ⚠️ **자동 처리 방법**: finance-manager API 사용
     - POST http://localhost:6003/api/excel-import (multipart, file=파일)
     - 로그인: admin / admin123 (POST /api/login → JWT 토큰)
     - 중복이면 409 응답 → 건너뜀
   - 입력 후 파일명을 `memory/excel-processed.json`에 기록
   - 처리 완료 시 재성님께 알림: "📊 [날짜] 피트니스/센터 매출 자동 입력 완료!"
   - **중복 방지**: 이미 처리된 파일 또는 409 응답은 건너뜀
