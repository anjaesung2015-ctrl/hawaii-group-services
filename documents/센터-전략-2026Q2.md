> 작성: 2026.04.05 | 화요일 직원 미팅용

## 📍 현재 상황

### 우리의 강점
- 실내 코트 3면 + 연습코트 1면 (UB에서 희소)
- 식당 + 마트 + 파티하우스 복합 운영
- 테니스 아카데미 (선수반/일반/동호회)

### 현실 인식
- **위치가 외곽** → 일부러 찾아와야 함
- 테니스 대회 = 수익 낮음 (선수들이 안 먹고 안 마심)
- **배구 대회 1일 = 800만₮+** (테니스 대회의 10배)
- 식당 매출이 코트 매출만큼 큼 (4/5 기준 53%)

### 4월 매출 데이터 (5일간)
| 항목 | 금액 |
|------|------|
| 총 수입 | 20,039,900₮ |
| 총 지출 | 7,057,933₮ |
| 순이익 | **12,981,967₮** |

4/5 배구대회 날: **코트 402만 + 식당 302만 + 마트 131만 = 835만₮**

## 🎯 전략 방향

### 핵심 한 줄
> **"테니스장"에서 "스포츠 이벤트 센터"로 리포지셔닝. 사람 수 × 체류 시간 = 매출**

테니스는 **브랜드이자 앵커**. 수익은 **단체 이벤트 + 식당 + 마트**에서 만든다.

### Phase 2: 5~6월 실행

#### 2-1. 기업 이벤트 유치
| 항목 | 내용 |
|------|------|
| 타깃 | UB 기업/은행/광업회사 단체 행사 |
| 내용 | 체육대회, 팀빌딩, 워크숍 |
| 가격 | 50인 기준 2,000,000~3,000,000₮ |
| 접근 | 총무/HR 담당자 직접 영업, 링크드인 |

**왜 기업인가?**
- 30~50명 단체로 옴
- 회사 법인카드 → 가격 민감도 낮음
- 연례 행사 → 1번 잡으면 매년 반복
- 식당 매출 폭발 (점심+저녁)

#### 2-2. 주말 정기 이벤트
매주 뭔가가 있는 장소로 만들기:
| 요일 | 이벤트 |
|------|--------|
| 토요일 오전 | 어린이 테니스 체험 (무료/저가) → 학부모 = 식당 고객 |
| 토요일 오후 | 동호회 배구/배드민턴 리그 |
| 일요일 | 가족 스포츠 데이 / 생일파티 |

#### 2-3. 피트니스 ↔ 센터 연계
- 피트니스 회원 400명 DB 활용
- "주말 센터 이벤트 참가 할인" 안내
- 피트니스에서 모객 → 센터로 유도
